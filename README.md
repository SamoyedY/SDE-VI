**ICML 2026 Spotlight**

This repo is forked from [ACSSM](https://github.com/bw-park/ACSSM), and the current implementation is developed based on that repository.

> Note: The codebase is still being cleaned up and reorganized.  
> A complete and polished version of the code will be released later.  

---

## Overview

Irregular time series are widely encountered in real-world applications, where observations arrive at non-uniform time intervals. This project studies generative modeling for irregular time series through an SDE-induced continuous-discrete variational inference framework.
